# Implementation Plan — SensorTelemetry

## 1. Общие сведения и задачи

Целью данного проекта является создание консольного инструмента на C++ для автоматизированной обработки и анализа телеметрических данных, хранящихся в формате JSON.

**Ключевой функционал системы:**

* **I/O операции:** чтение внешних JSON-файлов и генерация синтетических наборов данных (включая стресс-тесты с ошибками).
* **Data Validation:** проверка структуры JSON, типов полей и логической целостности (уникальность временных меток, диапазоны значений).
* **Аналитический блок:** расчет статистических показателей (min/max, медиана, среднее) и выявление аномалий методом скользящего окна.
* **Benchmarking:** измерение производительности алгоритмов при обработке больших массивов данных.

## 2. Технический стек и структура

**Инструментарий:**

* **Язык:** C++17 (MS Visual Studio).
* **Библиотека:** `nlohmann/json` для парсинга.
* **Контейнеры:** `std::vector` для данных, `std::set` для валидации уникальности `ts`.

**Архитектура проекта:**

* `/src` и `/include` — разделение интерфейсов (.h) и реализации (.cpp).
* `/data` — хранилище исходных и сгенерированных JSON-файлов.
* `/docs` — техническая документация и отчеты по производительности (Bench Report).

## 3. Дорожная карта разработки (Milestones)

### Модуль А: Подготовка и генерация данных

* Анализ схемы JSON: поля `sensor` (string), `ts` (ISO8601), `value` (double).
* Разработка генератора эталонных данных.
* Разработка генератора «грязных» данных (дубликаты, пропуски полей, выход за границы 0–100).

### Модуль Б: Ядро обработки (Processing Core)

* Интеграция парсера и маппинг JSON на внутренние структуры C++.
* **Валидация:** проверка типов и фильтрация некорректных записей.
* **Алгоритмы:** реализация эффективного вычисления медианы и оптимизированного скользящего окна (rolling window).

### Модуль В: Интерфейс и тестирование

* Форматированный вывод отчетов по конкретным сенсорам.
* Сбор метрик производительности при нагрузке (100, 1000, 10000 записей).
* Подготовка финального `.exe` файла и документации на GitHub.

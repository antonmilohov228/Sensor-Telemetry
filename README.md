# SensorTelemetry
##SensorTelemetry## — это консольный инструмент на C++, разработанный для полного цикла работы с телеметрией датчиков. Приложение позволяет автоматизировать процесс тестирования и обработки данных в формате JSON: от генерации тестовых наборов до глубокого статистического анализа.

Ключевые возможности:

Гибкая генерация: создание корректных и «зашумленных» JSON-файлов для проверки устойчивости систем.

Строгая валидация: проверка структуры и логической целостности данных с детальной диагностикой ошибок (`value`, `ts`,).

Статистический модуль: расчет экстремумов, среднего арифметического и медианы для выбранных датчиков (min, max, average, median).

Детекция аномалий: использование алгоритма скользящего окна для выявления отклонений в реальном времени.

Профилирование: встроенный мониторинг времени выполнения критических операций для оценки производительности (benchmarks).
---
##  Запуск программы

1. Скачай репозиторий и перейди в папку `release/`.  
2. Нажми на `SensorTelemetry.exe`.

---
Меню программы:
---

# SENSOR TELEMETRY


Создайте JSON файл с названием: data_{number}.json
Исчисление файлов начинается с нуля
Выберите действие:
1) Замер min, max, average, median и нахождения аномалий
2) Режим отладки
3) Выйти из программы
---
Чтобы выполнить анализ собственных данных, поместите ваши JSON-файлы в директорию `release/`. 
Образцы тестовых файлов расположены в папке `data/`.

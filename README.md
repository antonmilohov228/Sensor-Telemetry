SensorTelemetry — это консольный инструмент на C++, разработанный для полного цикла работы с телеметрией датчиков. Приложение позволяет автоматизировать процесс тестирования и обработки данных в формате JSON: от генерации тестовых наборов до глубокого статистического анализа.

Ключевые возможности:

Гибкая генерация: создание корректных и «зашумленных» JSON-файлов для проверки устойчивости систем.

Строгая валидация: проверка структуры и логической целостности данных с детальной диагностикой ошибок.

Статистический модуль: расчет экстремумов, среднего арифметического и медианы для выбранных датчиков.

Детекция аномалий: использование алгоритма скользящего окна для выявления отклонений в реальном времени.

Профилирование: встроенный мониторинг времени выполнения критических операций для оценки производительности.
